# DigiTools Purchase

## 📖 Description
It's a simple React project where user can purchase different types of digital products and tools. It includes a navbar, banner, statistics section, dynamic product listing, cart functionality, and a footer. The application is fully responsive for mobile devices.

## 🛠️ Tech Stack
- React.js
- JavaScript
- Tailwind CSS
- DaisyUI
- React-Toastify (NPM Package)
- JSON (for product data)

## 🚀 Features
- Toggle between Products and Cart views with dynamic UI updates
- Remove items from the cart with instant toast notifications
- Proceed to checkout clears the cart, shows a toast notification, and displays an empty cart UI
